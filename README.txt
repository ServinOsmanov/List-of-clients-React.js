1) You should download the files from my repository and save it in a directory on your PC
2) Download Node.js with npm included (https://nodejs.org/en/)
3) Install it
4) Open Node.js command prompt and write "cd" and the route to the previously created directory. For example "cd C:\library"
5) Write "node start.js" 
6) Open Node.js command prompt and choose the directory again
7) Write "npm start"
